## üì± About the Ejaar App

Ejaar is a mobile application that addresses the lack of digitalization in Bahrain's rental market. The app provides:

- **Direct Property Posting**: Landlords post properties directly to renters
- **Online Payments**: Secure payment processing
- **Issue Reporting**: Easy maintenance and issue tracking
- **User Profiles**: Separate interfaces for landlords and renters
- **Real-time Updates**: Live property availability

The mobile app is built with Flutter and Dart, providing a seamless cross-platform experience for both iOS and Android users.
